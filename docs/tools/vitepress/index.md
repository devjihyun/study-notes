# VitePress

> Vue 기반 정적 문서 사이트 생성기  
> 빠른 빌드 속도와 높은 커스터마이징 자유도를 제공하는 문서화 도구

## 1. 개요

VitePress는 Vite 기반으로 동작하는 **정적 사이트 생성기(SSG)**로  
개발 문서, 사내 위키, 가이드 사이트 구축에 적합합니다.

마크다운 기반 문서 작성 + Vue 컴포넌트 사용이 가능해  
**기술 문서 관리에 최적화**된 구조를 제공합니다.

[VitePress](https://vitepress.dev/ko/)

## 2. 주요 특징

- 마크다운 기반 문서 작성
- Vite 기반 빠른 로컬 서버
- Vue 컴포넌트 직접 사용 가능
- 다크모드 기본 지원
- 코드 하이라이팅
- 검색 기능 기본 제공
- 정적 사이트 생성(SSG)
- SEO 친화적 구조

## 3. 장점

- 초기 세팅이 간단함
- 빌드 속도가 빠름
- 테마 커스터마이징 자유도 높음
- GitHub 연동 쉬움
- 문서 + 실행 데모 페이지 구성 가능
- 유지보수 부담 적음

## 4. 활용 사례

- 개발 가이드 문서
- 사내 위키
- API 문서
- 기술 블로그
- 코드 실습 예제 포함 문서
- 개인 학습 자료 정리
